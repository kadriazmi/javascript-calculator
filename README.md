# JavaScript Calculator

A simple calculator built using **HTML, CSS, and JavaScript**.  
This project was created as part of my JavaScript learning journey.

The application allows users to perform basic arithmetic operations through an interactive interface.

---

## Features

- Basic arithmetic operations (+, −, ×, ÷)
- Interactive calculator buttons
- Real-time display of the calculation
- Persistent state using **localStorage**
- Clear button to reset the calculation

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Browser Local Storage
